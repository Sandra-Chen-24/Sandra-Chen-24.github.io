# 安裝 windows 版本 Gitlab runner 

```text
.\gitlab-runner.exe register  --url ${gitlab domain}$  --token ${token}
.\gitlab-runner.exe install
.\gitlab-runner.exe start
.\gitlab-runner.exe restart
.\gitlab-runner.exe stop
```
